# Hoopers Radar

App de basquete para criar perfil, encontrar parceiros e buscar quadras no mapa.

## Instalação

1. Abra um terminal em `d:\Nova pasta\HoopersRadar`
2. Execute `npm install`
3. Execute `npm run dev`

## Funcionalidades incluídas

- Cadastro de perfil de jogador local
- Exibição de parceiros recomendados
- Mapa de quadras usando OpenStreetMap

## Estrutura

- `src/App.tsx`: layout principal
- `src/components/ProfileForm.tsx`: formulário de perfil
- `src/components/PartnerList.tsx`: lista de parceiros
- `src/components/CourtMap.tsx`: mapa de quadras
